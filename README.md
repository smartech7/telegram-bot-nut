> The Telegram bot library that doesn't drive you nuts

## Installation

You can install the package via composer:

```bash
composer require sergix44/nutgram
```

## Usage

```php
$bot = new Sergix44\Nutgram('telegram-token-xxx');
echo $bot->sendMessage('hi!');
```

## Testing

```bash
composer test
```
